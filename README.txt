# EHR Quality Index Project with Recommendations

## الفكرة
مشروع لحساب جودة السجلات الصحية الإلكترونية مع توصيات ذكية لكل سجل ناقص.
يعطي لكل سجل درجة الجودة + نصائح لتحسين البيانات.

## طريقة التشغيل
1. تأكد من وجود Python + المكتبات pandas, matplotlib, openpyxl
2. افتح Terminal أو CMD داخل مجلد المشروع
3. شغل الكود:
   python scripts/ehr_quality.py
4. النتائج تظهر في ملف:
   results/quality_scores_with_recommendations.xlsx
